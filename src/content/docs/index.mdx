A simple and lightweight **REST API** built with Node.js and Express.js for managing movie data.

This API allows you to perform full **CRUD operations** on a movie dataset, including creating, retrieving, updating, and deleting movie records. It uses:

- An **in-memory data store** for fast access
- A **JSON file (`database.json`)** for persistence

---


## Features

- Create, update, and delete movie metadata  
- Search movies by title  
- Filter movies by genre  
- Pagination support for large datasets  
- Input validation using Joi  
- Interactive API documentation via Swagger  

---

1. Install dependencies  
2. Run the server  
3. Visit `/api-docs` to test endpoints
